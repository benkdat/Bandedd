# Banded — Compensation Intelligence Dashboard

A React-based compensation intelligence dashboard with filtering, salary band building, and trend visualization.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Then open http://localhost:5173 in your browser.

## Build for Production

```bash
npm run build
```

The built files will be in the `dist/` folder.

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "Import Project" and select your GitHub repo
4. Click "Deploy" — that's it!

## Tech Stack

- React 18
- Vite
- Recharts (charts)
- Lodash (utilities)
